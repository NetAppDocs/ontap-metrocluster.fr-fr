= Notes de version de MetroCluster
:allow-uri-read: 


Chaque version du logiciel de gestion des données ONTAP 9 inclut de nouvelles fonctionnalités améliorées qui améliorent les fonctionnalités, la gestion et les performances des configurations ONTAP MetroCluster.

Pour plus d'informations sur la prise en charge des commutateurs et de la plateforme matérielle, les problèmes connus et les limites dans toutes les versions de ONTAP 9, ou pour connaître les fonctionnalités introduites dans les versions antérieures à ONTAP 9.9.1, reportez-vous au https://library.netapp.com/ecm/ecm_download_file/ECMLP2492508["Notes de version de ONTAP 9"^]. Vous devez vous connecter avec votre compte NetApp ou créer un compte pour accéder aux notes de version.
