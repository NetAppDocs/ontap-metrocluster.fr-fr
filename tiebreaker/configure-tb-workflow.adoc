= MetroCluster Workflow de configuration du Tiebreaker
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Après avoir installé le logiciel Tiebreaker, vous pouvez ajouter des configurations MetroCluster, configurer les paramètres SNMP et commencer à surveiller votre environnement MetroCluster.

.A propos de ce flux de travail
Vous pouvez utiliser cette procédure pour configurer le logiciel Tiebreaker après son installation.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["Une seule"]link:concept_configuring_the_tiebreaker_software.html["Configurer le logiciel disjoncteur d'attache"]
[role="quick-margin-para"]
Ajoutez des configurations MetroCluster au logiciel Tiebreaker, importez les certificats et configurez les paramètres de surveillance.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["Deux"]link:task_configure_snmp_setts_for_tiebreaker_software.html["Configurer les paramètres SNMP pour le logiciel MetroCluster Tiebreaker"]
[role="quick-margin-para"]
Configurez les paramètres SNMP afin que le logiciel Tiebreaker puisse envoyer des notifications SNMP pour les événements MetroCluster.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["Trois"]link:concept_monitoring_the_mcc_configuration.html["Surveiller la configuration MetroCluster"]
[role="quick-margin-para"]
Utilisez le logiciel Tiebreaker pour surveiller l'état de la configuration MetroCluster, visualiser les événements SNMP et gérer les paramètres AutoSupport.
