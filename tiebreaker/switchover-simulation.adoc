= Simulez un basculement à l'aide de MetroCluster Tiebreaker
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
À partir de Tiebreaker 1.7, vous pouvez simuler un basculement pour tester la fonctionnalité de basculement de MetroCluster Tiebreaker.

.Description de la tâche
* Le `monitor switchover-simulate` Cette commande n'est prise en charge que dans Tiebreaker 1.7 ou version ultérieure.


.Étapes
. Vérifier l'état de surveillance du départage :
`monitor show -status`

. Effectuer une simulation de basculement :
+
`monitor switchover-simulate -monitor-name <monitor_name> -cluster <cluster_name>`

+
La commande renvoie le résultat suivant en cas de succès :
